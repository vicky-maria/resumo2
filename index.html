📙Imagens que se adaptam

Atualmente as pessoas acessam a internet por diferentes dispositivos: celular, computador, tablet e até mesmo televisão. Sendo assim, o desenvolvedor deve criar sites que se adaptem a diferentes tipos de tela, para que o usuário tenha uma boa experiência não importa qual dispositivo esteja usando.

Hoje, nós vamos falar sobre adaptação de imagens a diferentes tipos de tela. Se colocarmos uma imagem muito grande ela pode até ficar boa no computador, mas no celular vai ser bem incômodo, já que o usuário vai ter que ficar rolando a tela pra poder ver a imagem. Fora que arquivos de imagens muito grandes deixam o site lento, o que pode fazer com que o usuário não fique muito tempo no seu site.

📖 A tag <picture>

A tag <picture> muda a imagem que é exibida de acordo com o tamanho de tela. Por exemplo, se o tamanho da tela é de 300px ela vai exibir a imagem que tem 300px.Abaixo, vc vai ver o código e a definição de cada elemento.

    <picture>
        <source media"(min-width: 1000px)" srcset="imagens/imagemg.png" type="image/png">
        <source media"(min-width: 700px)" srcset="imagens/imagemm.png" type="image/png">
        <source media"(min-width: 300px)" srcset="imagens/imagemp.png" type="image/png">
        <img src="imagens/imagemg.png" alt="imagem">
        <img src="imagens/imagemm.png" alt="imagem">
        <img src="imagens/imagemp.png" alt="imagem">
    </picture>
📌 media
Define o tamanho necessário para que a imagem seja carregada

📌 srcset
Nome da imagem que vai ser carregada quando tiver o tamanho necessário

📌 type
Tipo da imagem

Aumente e diminua o navegador e veja a magia acontecer.
